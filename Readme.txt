Астероиды. Проект по Pygame(Яндекс Лицей)
Автор проекта: Алексей Овчинников

Идея: Основная идея моей игры взята со старой аркады под названием space shooter 90, но немного изменена сложность.

Реализация: Для начала необходимо было создать модели астероидов, пуль и т.д. Для этого я воспользвоался сайтом https://opengameart.org
	    С которого взял архив с необходимыми картинками. Также необходимо было найти подходящие звуки, что я сделал с помощью поиска в 
	    Интернете.

В проекте присутствуюь такие классы как:
	class Player
	class Mob
	class Bullet
	class Pow
	class Explosion
В классах выше реализован практически весь фунционал библиотеки pygame, а также весь необходимый фунцкионал для всех объектов и анимаций.
Помимо этого в проекте используется большое количество фунцкий(def):
	newmob
	draw_shiel_bar
	draw_lives
	show_go_screen
	draw_text
А также несколько циклов в конце. Функции отвечают за отрисовку некоторых объектов, которым не нужен отдельный класс.
В проекте реализована запись конечного счёта в базу данных и вывод лучшего счёта в текстовый файл.
файл best.db отвечает за запись результов игры, а best.txt за запись лучшего счёта.
в папках img и snd хранятся картинки и звуки соответственно.

Для запуска этой игры потребуется библиотека pygame, sqlite3 и random.


!!!ПРИМЕЧАНИЕ!!!:
 Управление осуществляется на клавиши A и D , огонь на пробел(space) - ракета перемещается вдоль оси Y, в левом верхнем углу указано здоровье игрока,
а справа сверху - количество оставшихся жизней(на старте их 3) сверху по середине указан текущий счёт игрока, замечу , что при закрытии игры до смерти 
игрока не повлечёт никаких изменений в базу данных и текстовый файл. Также замечу, что в базе данных лучше всег оставлять одно значение, иначе программа
может работать не корректно. Чтобы игра заработала необходимо запустить файл Asteroids.py
